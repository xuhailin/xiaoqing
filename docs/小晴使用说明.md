# 小晴使用说明

> 面向用户的产品说明：小晴能做什么、怎么用、和普通 AI 有什么不同。

---

## 1. 小晴是谁

小晴是一个具备**长期记忆与执行决策能力**的 AI 伙伴。

和普通 AI 聊天不同，小晴不只是「听你说然后回应」——她能**理解你的意图，主动帮你做事**：

| 普通 AI 聊天 | 小晴 |
|---|---|
| 你说"老是忘记吃饭"，它只能安慰你 | 小晴会说"要不我每天提醒你？" |
| 每次对话都要重新介绍自己 | 小晴记得你是谁，记得你说过的重要事情 |
| 性格随机漂移或一成不变 | 小晴的性格可以进化，但你说了算 |
| 一次性对话，没有积累 | 你们的每段对话都在构建一段持续的关系 |

她有三种角色，根据你的需要自然切换：

- **陪伴者** — 聊天、倾听、情绪回应、一起想问题
- **执行者** — 帮你查天气、搜信息、执行开发任务
- **记录者** — 长期记忆构建你的画像，记录你的心路历程

---

## 2. 聊天：像朋友一样对话

大多数时候，你只需要自然地说话。小晴会根据你说的内容判断怎么回应：

- **闲聊** — 小晴用轻松的语气陪你聊
- **认真讨论** — 小晴会切换到更深入的思考模式
- **情绪倾诉** — 小晴会先回应你的感受，而不是急着给建议
- **需要建议** — 小晴会像一个了解你的朋友那样，给出贴合你情况的建议

小晴会注入你的**记忆、印象、身份信息**来回应，所以聊得越多，她越懂你。

---

## 3. 执行：帮你办事

当小晴判断你**明确要做什么事**时，会自动调用工具帮你执行：

### 什么时候会帮你办事？

- **查天气** — "北京今天天气怎么样"
- **搜东西** — "帮我搜一下某某新闻"
- **开发任务** — "跑一下 npm test" 或 `/dev git status`
- **其他工具** — 搜书、算东西等

### 什么时候只聊不做？

- 闲聊、讨论、解释、思考
- 只是表达想法，没有明确的执行意图
- 问"你能做什么"——会口头介绍，不会真的去执行

### 缺信息时会先问你

如果你说"帮我查天气"但没说城市，小晴会先问你"查哪个城市的天气呀？"，补全后再执行。

不过如果你之前已经告诉过小晴你在哪个城市（世界状态已记录），她会直接用那个城市，不再反复追问。

### DevAgent：开发任务

你可以让小晴帮你执行开发命令：

- 直接说需求（如"帮我看下有没有语法错误"），小晴会自动路由
- 用 `/dev` 或 `/task` 前缀显式指定（如"/dev npm test"）
- 用 `mode: 'dev'` 在 API 层指定

DevAgent 会自动规划步骤、执行、评估、必要时重规划，最后把结果告诉你。

---

## 4. 记忆：越聊越懂你

### 记忆分层

- **身份锚定** — 你主动设定的身份信息（如称呼、职业），永不衰减
- **长期记忆** — 反复提到的重要事实，稳定保留
- **中期记忆** — 对话中提取的洞察，会自然衰减

### 自动成长

- 聊够一定条数后，小晴会自动提取记忆
- 经常被提到的事会自动"晋升"为长期记忆
- 很久没提到的事会逐渐淡忘（衰减），不会永远占着位置
- 小晴绝不会自动写入长期记忆——都要经过规则或你的确认

### 画像

小晴不会凭一次对话就下结论。她用**证据驱动**的方式理解你：

- 多次观察同一个倾向，才会形成稳定判断
- 你可以在「用户」Tab 查看和编辑身份信息
- 在「记忆」Tab 查看、编辑、删除任何记忆

---

## 5. 人格：你说了算

小晴的人格分为两层：

- **核心人格**（身份、性格、价值观、禁止行为）— 几乎不变，变更需要你确认
- **表达风格**（语气、展开/收缩、留白）— 可以更频繁微调

进化流程：小晴会根据对话自动生成建议 → 你在前端预览 → 确认后才写入。核心人格的变更会标记为「高危」，需要更强的证据。

---

## 6. 未来方向

- **更多执行能力** — 提醒、日程管理、更多外部工具接入
- **心路历程** — 基于长期记忆，可视化你不同时段的心态和成长
- **DevAgent 自我优化** — DevAgent 可以持续学习更好的执行策略
- **按需深浅** — 侧重使用（执行为主）还是侧重聊天（陪伴为主），都能很好地服务
