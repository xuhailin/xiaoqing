---

## 4.1 模式统一（重点）

当前：
- 文生视频
- 图生视频
- 首尾帧控制

👉 改为：

### 创作模式（统一入口）

- 文本创作（默认）
- 图片驱动（上传图）
- 分镜创作（开启分镜模式）

---

## 4.2 分镜设计（关键点）

分镜不是新页面，而是“增强模式”

实现方式：

- 默认：单段 prompt → 单视频
- 开启分镜：
  - 自动拆分为多个 scene
  - 每个 scene 独立 prompt
  - 共享风格 / 角色 / 设定

---

### 分镜交互要求：

- 可以自动拆（AI）
- 可以手动编辑每一镜
- 支持增删镜头
- 每一镜：
  - 文本 prompt
  - 时长
  - 镜头描述（可选）

---

## 4.3 UI结构建议

创作页结构：

- 顶部：模式切换（文本 / 图片 / 分镜）
- 中间：
  - prompt 输入
  -（分镜模式 → scene 列表）
- 右侧：
  - 风格控制（可复用 studio 的 package）
- 底部：
  - 生成按钮
